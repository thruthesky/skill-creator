# 워크플로우 패턴

## 순차적 워크플로우

복잡한 작업의 경우, 작업을 명확하고 순차적인 단계로 나누세요. SKILL.md 시작 부분에서 Claude에게 프로세스 개요를 제공하는 것이 종종 도움이 됩니다:

```markdown
PDF 양식 작성은 다음 단계를 거칩니다:

1. 양식 분석 (analyze_form.py 실행)
2. 필드 매핑 생성 (fields.json 편집)
3. 매핑 검증 (validate_fields.py 실행)
4. 양식 작성 (fill_form.py 실행)
5. 출력 확인 (verify_output.py 실행)
```
